# LR4-set-calculator
Интерактивный калькулятор множеств на языке Python, реализующий основные операции теории множеств:

Объединение (∪), пересечение (∩), разность (-), симметрическая разность (Δ), равенство (=)

Чтение множеств из файлов

Связать операции с реляционной алгеброй (теоретико-множественная основа БД)
